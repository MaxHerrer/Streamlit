# Predicción y Segmentación de Pacientes con Riesgo de Alzheimer

## Descripción del Proyecto

Este proyecto desarrolla una aplicación interactiva en Streamlit para apoyar el análisis de pacientes mediante técnicas de Machine Learning aplicadas a datos relacionados con la enfermedad de Alzheimer.

La aplicación integra tres modelos desarrollados durante el proyecto:

- **K-Means** (Aprendizaje No Supervisado): Segmentación de pacientes en grupos con características similares.
- **Árbol de Decisión** (Aprendizaje Supervisado): Clasificación de pacientes mediante reglas de decisión interpretables.
- **Regresión Logística** (Aprendizaje Supervisado): Estimación de la probabilidad de diagnóstico de Alzheimer.

## Objetivo

Proporcionar una herramienta interactiva que permita:

- Ingresar información de pacientes.
- Realizar predicciones mediante modelos supervisados.
- Identificar segmentos de pacientes mediante clustering.
- Visualizar resultados para apoyar la toma de decisiones basada en datos.

## Tecnologías Utilizadas

- Python

- Scikit-learn
- Matplotlib
- Plotly

## Modelos Implementados

### K-Means
Modelo de aprendizaje no supervisado utilizado para identificar grupos de pacientes con características similares, facilitando la detección de patrones y perfiles clínicos.

### Árbol de Decisión
Modelo de clasificación que genera reglas fácilmente interpretables para predecir la presencia o ausencia de Alzheimer a partir de variables clínicas y demográficas.

### Regresión Logística
Modelo de clasificación que estima la probabilidad de diagnóstico de Alzheimer y permite analizar la influencia de las variables sobre la predicción.

## Resultados

La aplicación permite:

- Obtener predicciones en tiempo real.
- Clasificar pacientes mediante modelos supervisados.
- Identificar segmentos de pacientes mediante clustering.
- Visualizar hallazgos relevantes para apoyar la toma de decisiones clínicas y de investigación.
